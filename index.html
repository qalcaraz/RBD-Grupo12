Sección #1 AWS

Tabla de Servicios de AWS

Hemos recopilado las descripciones de los servicios esenciales de AWS, incluyendo 6 servicios adicionales a los 8 propuestos.

Servicio Descripción
EC2 Servicio básico de la nube AWS. En EC2 se crean las máquinas virtuales en la nube. Permite su creación de forma rápida. Además, si la máquina virtual se queda corta, se puede reescalar añadiendo más potencia fácilmente.
RDS (Asignado a Elisa) Servicio de bases de datos relacionales gestionado que facilita la configuración, operación y escalado de una base de datos en la nube. Permite utilizar motores conocidos como MySQL, PostgreSQL u Oracle sin preocuparse por la administración del sistema operativo subyacente.
S3 (Asignado a Elisa) Servicio de almacenamiento de objetos con alta durabilidad y escalabilidad. Permite guardar cualquier tipo de archivo (fotos, videos, copias de seguridad) en "buckets" (cubos), siendo una solución ideal para almacenamiento web, archivado de datos y data lakes.
EBS (Asignado a Elisa) Proporciona volúmenes de almacenamiento persistente a nivel de bloque para usar con instancias EC2. Es como un disco duro virtual que se conecta a la máquina virtual, ideal para sistemas operativos, bases de datos transaccionales y cualquier aplicación que requiera I/O de baja latencia.
DynamoDB (Asignado a Elisa) Base de datos NoSQL clave-valor y de documentos rápida y flexible, diseñada para aplicaciones que necesitan un rendimiento constante con cualquier escala. Es totalmente gestionada, por lo que no hay que preocuparse por el escalado ni el mantenimiento del hardware.
Cloudfront (Asignado a Elisa) Red de distribución de contenido (CDN) que acelera la entrega de datos, videos, aplicaciones y APIs a nivel mundial. Almacena temporalmente el contenido cerca de los usuarios finales, reduciendo la latencia y mejorando la velocidad de carga de la web.
Amazon EKS (Asignado a Tomás) Servicio gestionado de Kubernetes que facilita la ejecución de aplicaciones en contenedores en AWS. Automatiza la administración y el escalado del plano de control de Kubernetes, permitiendo centrarse únicamente en el despliegue de las aplicaciones.
AWS Lambda (Asignado a Tomás) Servicio de computación sin servidor (serverless) que ejecuta código en respuesta a eventos sin necesidad de aprovisionar o gestionar servidores. Se utiliza para tareas de backend, procesamiento de datos en tiempo real y microservicios, pagando solo por el tiempo de cómputo consumido.
VPC (Servicio Añadido - Tomás) Permite crear una red virtual aislada en la nube de AWS donde se pueden lanzar recursos. Es como tener tu propio centro de datos dentro de AWS, donde controlas la configuración de red, subredes, tablas de ruteo y puertas de enlace.
Route 53 (Servicio Añadido - Tomás) Servicio de DNS (Sistema de Nombres de Dominio) web escalable y altamente disponible. Es el servicio que traduce el nombre de un dominio (ej: `www.midominio.com`) a la dirección IP de tu servidor, además de ofrecer registro de dominios y enrutamiento inteligente.
CloudWatch (Servicio Añadido - Kike) Servicio de monitoreo y gestión de observabilidad que recopila datos de rendimiento y operación de los recursos de AWS y las aplicaciones. Permite establecer alarmas que se activan cuando ciertas métricas (como el uso de CPU) superan un umbral.
AWS IAM (Servicio Añadido - Kike) Identity and Access Management es el servicio que gestiona de forma segura el acceso a los recursos de AWS. Permite crear usuarios, grupos y roles para definir quién puede hacer qué en vuestra cuenta, siendo fundamental para la seguridad y el cumplimiento de permisos.
SNS (Servicio Añadido - Kike) Simple Notification Service es un servicio de mensajería que permite desacoplar los sistemas de envío y recepción de información. Se utiliza para enviar notificaciones a varios suscriptores (como correos, SMS o funciones Lambda) a partir de eventos o alarmas.
Security Hub (Servicio Añadido - Kike) Ofrece una vista consolidada de los hallazgos de seguridad en todas las cuentas de AWS. Recopila alertas de varios servicios de seguridad (como GuardDuty e IAM Access Analyzer) y las presenta en un formato que facilita priorizar y responder a las amenazas.
